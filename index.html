Infinity Stone Collector

Rules:

The inifinity stones you see are each assigned a value. To start the game, please click one of the stones, which will generate a target number. By clicking the stones you will increase the counter by that stones value to try and reach the target number. If you reach that number, the win counter will go up and the game will restart. If you lose, Thanos destroys the entire universe.
